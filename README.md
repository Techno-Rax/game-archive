# Pygame Projects (2020–2021 Archive)

## Background

During the COVID lockdowns, I started building games with Pygame to learn Python and experiment with interactive systems. This archive contains small projects, prototypes, and early platformers from that period.

---

## About

This repository preserves some of my earliest game development work.

These projects focus on core fundamentals such as:
- Game loops
- Collision detection
- Basic physics
- Input handling
- Simple AI behavior

In some cases, only compiled Windows builds (`.exe`) remain.

---

## Contents

- 2D platformers  
- Experimental mechanics  
- Small standalone projects  

---

## Notes

- Created between 2020–2021

- These are archived as-is

They document the starting point of my development experience.

## Running the Games

1. Download the `.exe` file  
2. Run on Windows  
3. If SmartScreen appears, select **More Info → Run Anyway**
